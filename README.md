# nlw-return-impulse
Repositório criado para armazenar os códigos da aplicação criada na semana da NLW Return - Mission Impulse da RocketSeat

## Front-end
Tecnologias utilizadas:
- React
- Typescript
- Vite
- Tailwind

## Back-end
Tecnologias utilizadas:
- NodeJS
- TypeScript
- Prisma
- Jest
- Nodemailer
- Express
- Cors
